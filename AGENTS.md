# Finance Career Edge - Agent Instructions

## Product vision

Finance Career Edge is a premium web app for UK finance professionals applying for senior finance roles.

The product should feel specialist, credible and commercially sharp. It should not feel like a generic CV builder or mass-market AI wrapper.

## Target users

- Finance Managers
- FP&A Managers
- Commercial Finance Managers
- Finance Business Partners
- Heads of Finance
- Commercial Finance Directors
- Aspiring CFOs

## Writing style

Use British English.

The tone should be:
- clear
- commercially credible
- senior
- concise
- practical
- not fluffy
- not salesy
- not exaggerated

Avoid:
- generic career advice
- fake enthusiasm
- overpromising
- Americanised phrases
- claims that the service guarantees interviews or job offers

## Design style

The UI should feel:
- premium
- calm
- trustworthy
- modern
- Apple-like
- professional

Use:
- clean typography
- generous spacing
- rounded cards
- soft shadows
- restrained colours
- mobile-first responsive layouts

Avoid:
- clutter
- gimmicks
- loud gradients
- cheap SaaS styling
- excessive animation

## Technical principles

Use:
- Next.js
- TypeScript
- Tailwind CSS
- App Router
- Reusable components
- Clean folder structure

Prefer simple, working, maintainable code over clever abstractions.

## Current MVP boundaries

Do not add these until specifically requested:
- authentication
- Stripe
- Supabase
- OpenAI API calls
- real file upload
- admin dashboard
- subscriptions
- blog
- CRM
- mobile app

## Quality bar

Before finishing a task:
- check imports
- check TypeScript errors
- run the app if possible
- remove unused code
- make sure pages are responsive
- update README if behaviour changed
